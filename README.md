# 🎂 Happy Birthday Website ❤️

A simple yet heartfelt birthday website created for my girlfriend using **HTML**, **CSS**, and **JavaScript**. This project is designed to celebrate a special day with personalized messages, cherished memories, and interactive surprises.

## ✨ Features

- 🎁 Welcome birthday greeting
- 📹 Birthday video (her self)
- 💌 Personalized love letter
- ⏳ Relationship day counter
- 🎉 Surprise message

## 🛠️ Built With

- HTML5
- CSS3
